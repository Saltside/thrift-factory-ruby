# Thrift::Factory

Recursive [thrift][] struct factory.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'thrift-factory'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install thrift-factory

## Testing

First install `thrift` compilier on your platform.

	$ make test
